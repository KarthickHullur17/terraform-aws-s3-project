# 🚀 Terraform AWS S3 Bucket Project

## 📌 Overview

This project demonstrates how to create and manage an AWS S3 bucket using Terraform.

---

## 🛠️ Tech Stack

* Terraform
* AWS S3
* AWS IAM

---

## ⚙️ What This Project Does

* Creates an S3 bucket in AWS
* Enables versioning for the bucket
* Uses Infrastructure as Code (IaC)

---

## 🚀 Commands Used

```bash
terraform init
terraform plan
terraform apply
```

---

## 📊 Output

* S3 bucket created successfully
* Versioning enabled
* Bucket name displayed

---

## 📈 Resume Point

Provisioned AWS S3 bucket using Terraform with versioning enabled, demonstrating Infrastructure as Code (IaC) skills.
